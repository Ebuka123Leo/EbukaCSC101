	\subsection{HISTORY OF BASIC PROGRAMMING}
	\paragraph{The BASIC programming language, which also stands for "Beginners ALL SYMBOLIC INSTRUCTIONS CODE", was invented in the year 1964 in Dartmouth college ,US. This programming language was invented by two Dartmouth mathematicians called John George Kemeny and Tom Kurtzas. They both had the aim to make computers very easy to use and availabilty of computing devices to everyone in their college.
		\paragraph{This programming language helps students to learn and write simple computer programs in order to use them to create advanced computer programs}
